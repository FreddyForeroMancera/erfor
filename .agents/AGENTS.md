# Reglas del Proyecto ERFOR

- **Auto-Commit y Despliegue**: Siempre que se cumpla una acción o tarea solicitada por el usuario que implique cambios en el código, DEBES ejecutar automáticamente un `git add .`, `git commit -m "..."` y `git push origin main`. Esto es requerido para que los cambios se desplieguen inmediatamente a Vercel a través de GitHub, sin esperar a que el usuario lo solicite explícitamente.
