# Reglas del Proyecto ERFOR

- **Commits y Despliegues Manuales**: A partir de ahora, **NO** debes hacer auto-commit ni push automático después de cada acción (a diferencia de lo establecido inicialmente). Los commits y los pushes hacia Vercel solo se realizarán de manera premeditada y explícita cuando el usuario te lo solicite, esto con el fin de evitar el consumo innecesario de recursos o tokens en Vercel.
